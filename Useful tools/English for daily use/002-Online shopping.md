# Online shopping
夏天到了，你很想买一条 dress 裙子

but 你不知道今年流行什么，所以你就去刷 social media 社交媒体

看那些 fashion influencers穿搭博主 去得到 the latest trends 最流行的趋势

找呀找，终于找到了一条特别漂亮的裙子，然后点开 comments section评论区 

发现博主特别贴心的 tag the brand name 标注了品牌名

你点开了 official store 官方店铺 

你被他们的价格吓了一跳，你觉得卖家 overpriced 要价太高了

这里可能有一个 a high brand premium 很高的品牌溢价

not cost effective\ not worth the price 性价比一般\不值那个价

search for a dupe 上网找个同款 

发现刚好有一个 indentical design 设计一样的

similar looking 外观相似

affordable price 价格你付得起

check the review 就看了一下他们的买家秀 

特别是那些有 real photo 的

你觉得还可以，就问了一下 customer service 客服

问 你们是 available now\ in stock 现货 ready to ship 现在就可以发货吗

还是 pre order  预售

你们的 estimated delivery time 预计物流时间是多少

客户回复same-day delivery 当日达到 

next-day delivery 次日送达

standard delivery\shipping 普通快递

你觉得可以，就place your order 下单了

等着 package 包裹 送过来

到了之后，立刻 try on 试穿

发现 a good fit 很合体

very slimming 显瘦

fabric 材质也不错

准备 keep it 留下来 

反之，你发现  fabric  papery\cheap 材质不好，像纸一样、廉价

makes u look bigger 显胖

too loose 太松 too tight 太紧

apply for return 决定退掉

申请退款之后，得到 return code 退款编码

把return code 写在 package 或者 parcel 上

等着express\ delivery\courier 快递员 pick up 带走
